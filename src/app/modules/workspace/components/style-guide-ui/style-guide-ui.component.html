Buttons

We have 3 variants on buttons Solid(primary buttons), Bordered(secondary buttons), Transparent(tertiary buttons)

Primary buttons
button button button
Secondary buttons

Button Type Class Mixin
Primary button .cta-primary 
@include mixin-name(background-color, text-color);
@include button-solid($primary-color1, $white-color);
Primary button with left icon NA 
@include mixin-name(background-color, text-color, icon-path, icon-position);
@include button-solid($primary-color1, $white-color, '/assets/images/icon-name.svg', left);
Primary button with right icon NA 
@include mixin-name(background-color, text-color, icon-path, icon-position);
@include button-solid($primary-color1, $white-color, '/assets/images/icon-name.svg', right);
Secondary button .cta-secondary 
@include mixin-name(border-and-text-color);
@include button-brd($primary-color1);
Secondary button with left icon NA 
@include mixin-name(border-and-text-color, icon-path, icon-position);
@include button-brd($primary-color1, '/assets/images/icon-name.svg', left);
Secondary button with right icon NA 
@include mixin-name(border-and-text-color, icon-path, icon-position);
@include button-brd($primary-color1, '/assets/images/icon-name.svg', right);
Tertiary button .cta-tertiary 
@include mixin-name(color);
@include button-transparent($primary-color1);
Tertiary button with left icon NA 
@include mixin-name(color, icon-path, icon-position);
@include button-transparent($primary-color1, '/assets/images/icon-name.svg', left);
Tertiary button with right icon NA 
@include mixin-name(color, icon-path, icon-position);
@include button-transparent($primary-color1, '/assets/images/icon-name.svg', right);

Note: Even we can use image tag for icon on buttons with out any mixin dependency
